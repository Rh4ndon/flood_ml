About This System

A localized GIS-based platform to predict and visualize weather conditions in Bongabon, Palayan, and Cabanatuan.

Located in Nueva Ecija — a province prone to heavy monsoon rains and seasonal flooding — these three municipalities face significant weather-related risks that impact agriculture, transport, and daily life.

This system leverages machine learning and spatio-temporal GIS mapping to provide hyperlocal, real-time weather predictions — helping residents, local governments, and emergency responders prepare effectively.

Predictions are generated using local weather data and displayed on an interactive map, with color-coded markers indicating “Good” or “Adverse” weather conditions for each of the three towns.

Frequently Asked Questions

  • What problem does this solve?

    Bongabon, Palayan, and Cabanatuan face frequent weather disruptions — from heavy rains to seasonal flooding — that impact agriculture, transport, and safety. This system provides localized, real-time weather predictions to help residents and local government units make informed decisions — from daily planning to disaster preparedness.

  • How was this built?

    DATA
    We trained our model using historical and real-time weather data (temperature, wind, precipitation, humidity, cloud cover) specifically for Bongabon, Palayan, and Cabanatuan.

    MODEL
    A Decision Tree Classifier (via scikit-learn) predicts “Good” or “Adverse” weather. The model is serialized via pickle and loaded in Flask.

    GIS MAPPING
    Using Folium, we plot predictions on an interactive map focused on these three towns. Each location is marked with color-coded pins (green = good, red = adverse) and includes a temperature heatmap layer.

    FRONTEND
    Built with Flask, Bootstrap, and vanilla JavaScript. Hosted for public access.
